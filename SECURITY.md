# Security Policy

See [Heretek-RE Security Policy](https://github.com/Heretek-RE/.github/blob/main/SECURITY.md) for the full policy.

## Reporting

Use [GitHub Security Advisories](https://github.com/Heretek-RE/re-triton/security/advisories/new) to report vulnerabilities privately. We acknowledge receipt within 7 days and aim to issue a fix or mitigation within 30 days for high-severity issues.
